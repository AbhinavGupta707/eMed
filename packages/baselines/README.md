# Derived personal baselines

This package builds versioned, within-person projections from quality-passing structured facts.
Provider, provider version, algorithm, device class, platform, and capture surface must match before
samples are compared. Unknown context and non-comparable history remain explicit outcomes.

A `changed` projection is descriptive personal context only. It is not a diagnosis, a population
threshold, a quality decision, or workflow authority. Raw camera, audio, video, and conversational
content are outside the schemas by design.
