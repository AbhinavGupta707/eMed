# Requirements and scenario traceability

## Status vocabulary

- `PASS`: observed evidence satisfies the narrow stated local/no-key claim.
- `PARTIAL`: some required evidence exists, but a named external/manual portion remains open.
- `PENDING`: the required evidence was not observed; release needs the named waiver accepted or the closure action completed.
- `DEFERRED`: explicitly outside the hackathon release scope and not claimed.

Waiver IDs refer to the [open waiver/risk register](./release-checklist-and-waivers.md). An open waiver is not approval and never converts missing evidence into a pass.
